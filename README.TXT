========================
SQUAT 1.0, http://squatgrid.sourceforge.net
========================

Description of the files contained in this distribution:

- lib/*.jar: folder containing all libraries needed to execute SQUAT
- initsquat-1.0.jar: executable jar file to launch SQUAT
- src.zip: source code
- README.TXT: this file


To run the project from the command line, go to the dist folder and
type the following:

java -jar "initsquat.jar" 

Otherwise just double click on initsquat-1.0.jar executable jar file.
